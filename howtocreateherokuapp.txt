***Request Response Cycle:
1)user types url of the app (later provided when you git your code to Heroku)
2)browser issues a request for that url from routes.py
3)request hits routes.py where it pass it to a Python function
4)Python function finds the correct html in templates folder
5)the html file gets info from css, js, img in static folder
6)rendered html is sends back to routes.py
7)routes.py sends back requested html to browser->page loads
